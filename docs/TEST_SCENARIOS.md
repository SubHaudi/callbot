# 🧪 Callbot 테스트 시나리오

데모 페이지에서 음성/텍스트로 테스트할 수 있는 시나리오 모음.

> **Demo**: https://da0dhxbqgiqxc.cloudfront.net/demo

---

## 🔑 기본 정보 (테스트용)

| 항목 | 값 |
|------|-----|
| 전화번호 | 010-1234-5678 |
| 생년월일 | 901215 |

> 본인 확인이 필요한 기능은 전화번호 → 생년월일 순으로 인증 진행

---

## 시나리오 1: 요금 조회 (BILLING_INQUIRY)

**목적**: 가장 기본적인 플로우 — 인증 → 조회 → 응답

```
👤 "요금 조회해주세요"
🤖 (본인 확인 요청 — 전화번호)
👤 "010-1234-5678"
🤖 (인증 요청 — 생년월일)
👤 "901215"
🤖 (요금 정보 응답: 요금제명, 금액 등)
```

**체크포인트**:
- [ ] 본인 확인 절차 정상 진행
- [ ] PII 마스킹 작동 (로그에서 전화번호 마스킹 확인)
- [ ] 요금 정보가 자연스러운 문장으로 나오는지

## 시나리오 2: 납부 확인 (PAYMENT_CHECK)

```
👤 "이번 달 요금 냈는지 확인해주세요"
🤖 (본인 확인 절차)
👤 (전화번호 + 생년월일)
🤖 (납부 상태 응답)
```

**체크포인트**:
- [ ] "납부 완료" 또는 "미납" 상태 정상 표시

---

## 시나리오 3: 요금제 조회 (PLAN_INQUIRY)

```
👤 "어떤 요금제 있어요?"
🤖 (가입 가능한 요금제 목록 안내)
```

**체크포인트**:
- [ ] 인증 없이 바로 응답하는지 (조회는 인증 불필요할 수 있음)
- [ ] 요금제 목록이 나오는지

---

## 시나리오 4: 요금제 변경 (PLAN_CHANGE)

```
👤 "요금제 변경하고 싶어요"
🤖 (본인 확인 절차)
👤 (전화번호 + 생년월일)
🤖 (현재 요금제 안내 + 변경 가능 목록)
👤 "5G 프리미어로 바꿔주세요"
🤖 (변경 확인 요청)
👤 "네"
🤖 (변경 완료 안내)
```

**체크포인트**:
- [ ] 멀티스텝 대화 정상 진행 (현재 → 선택 → 확인 → 완료)
- [ ] 변경 전 확인 절차 있는지

## 시나리오 5: 데이터 잔여량 조회 (DATA_USAGE_INQUIRY)

```
👤 "데이터 얼마나 남았어요?"

🤖 (잔여 데이터량 응답)
```

- [ ] GB 단위로 자연스럽게 안내

## 시나리오 6: 부가서비스 해지 (ADDON_CANCEL)

```
👤 "부가서비스 해지하려고요"
🤖 (본인 확인 절차)
👤 (전화번호 + 생년월일)
🤖 (현재 부가서비스 목록 안내)
👤 "컬러링 해지해주세요"
🤖 (해지 확인 + 완료)
```

**체크포인트**:
- [ ] 부가서비스 목록 표시
- [ ] 해지 확인 절차

---

## 시나리오 7: 상담사 연결 (AGENT_CONNECT)

```
👤 "상담사 연결해주세요"
🤖 (상담사 연결 안내 — 그룹 라우팅)
```

**체크포인트**:
- [ ] 즉시 에스컬레이션 처리
- [ ] 적절한 상담 그룹으로 라우팅 안내

---

## 시나리오 8: 해지 문의 (CANCELLATION) — 에스컬레이션

```
👤 "해지하고 싶어요"
🤖 (상담사 연결 안내 — 해지/불만 상담 그룹)
```

**체크포인트**:
- [ ] 봇이 직접 처리하지 않고 상담사로 넘기는지
- [ ] ESCALATION_INTENTS 정상 작동

---

## 시나리오 9: 불만 접수 (COMPLAINT) — 에스컬레이션

```
👤 "요금이 이상하게 나왔어요, 항의하려고요"
🤖 (공감 표현 + 상담사 연결 안내)
```

**체크포인트**:
- [ ] 감정 인식/공감 표현
- [ ] 상담사 에스컬레이션

---

## 시나리오 10: 일반 문의 (GENERAL_INQUIRY)

```
🤖 (일반 안내 응답)
```

**체크포인트**:
- [ ] LLM 기반 자연어 응답

## 🎙️ 음성 특화 테스트

### 시나리오 V1: Barge-in (끼어들기)

```
🤖 (긴 요금제 설명 중...)
👤 (중간에 말을 끊고) "아 그만, 요금만 알려줘"
```

**체크포인트**:
- [ ] TTS 재생 중단
- [ ] 새 발화 인식 + 인텐트 전환

### 시나리오 V2: 속도 조절

```
👤 "좀 천천히 말해주세요"
```

**체크포인트**:
- [ ] SPEED_CONTROL 인텐트 인식
- [ ] TTS 속도 변경 (또는 안내)

### 시나리오 V3: 다시 말해줘

```
👤 "다시 한번 말해주세요"
```

**체크포인트**:
- [ ] REPEAT_REQUEST 인텐트 인식
- [ ] 이전 응답 재생

### 시나리오 V4: 통화 종료

```
👤 "끊을게요" / "감사합니다 끊겠습니다"
```

**체크포인트**:
- [ ] END_CALL 인텐트 인식
- [ ] 종료 인사 + 세션 정리
